---
description: Remove items from an entity's inventory
---

# Clear Item

```yaml
clear_item:
  item: my_item
  amount: 2 # Optional (Default value: 1)
```

For `item`, refer to [this link](https://jd.papermc.io/paper/org/bukkit/inventory/ItemType.html#field-summary) for the
value or a custom item with `namespace:value` or `value`
