---
title: Behaviours
description: Add custom logic to blocks, furniture, and complex furniture via YAML
icon: Zap
---

Behaviours add persistent, event-driven logic to your custom blocks and furniture.
They are configured under the `behaviours` key on any ItemsAdder item.

```yaml
items:
  my_block:
    # ...
    behaviours:
      contact_damage:
        amount: 1.0
```
