---
title: Campfire & Stonecutter Recipes
description: Add support for Campfire and Stonecutter recipes (requires 1.21.11+)
---

<Callout type="warn">
  **Version Requirement:** Requires **Paper 1.21.11 or higher**. The feature is disabled on older versions.
</Callout>

## Campfire Cooking

```yaml
info:
  namespace: some_namespace

recipes:
  campfire_cooking:
    my_roasted_item:
      enabled: true # Optional (Default value: true)
      ingredient:
        item: my_item:my_custom_item
      result:
        item: my_item:my_cooked_custom_item
        amount: 2 # Optional (Default value: 1)
      cook_time: 200 # Optional (Default value: 600) - Value in ticks (20 ticks = 1 second)
      exp: 0.1 # Optional (Default value: 0.0)
```

## Stonecutter

```yaml
info:
  namespace: some_namespace

recipes:
  stonecutter:
    my_cut_block:
      enabled: true # Optional (Default value: true)
      ingredient:
        item: my_item:my_custom_stone
      result:
        item: my_item:my_cut_custom_stonemy_custom_item
        amount: 4 # Optional (Default value: 1)
```
