## `/itemsadderadditions`

Main administrative command for ItemsAdderAdditions.

Aliases:

```text
/iaadditions
/iaa
```

## Permissions

Base permission:

```text
itemsadderadditions.command.itemsadderadditions
```

## Subcommands

### `/itemsadderadditions reload`

Reloads ItemsAdderAdditions `config.yml` and refreshes plugin-level feature, action, behaviour, and component toggles.

```text
/itemsadderadditions reload
```

Permission:

```text
itemsadderadditions.command.itemsadderadditions.reload
```

This command does **not** reload ItemsAdder content YAML files.

Use `/iareload` after changing ItemsAdderAdditions content YAML sections such as actions, behaviours, custom tags, recipes, advancements, custom paintings, Valhalla metadata, or world generation entries.

Use `/iazip` after changing `components` sections or resource-pack assets.

### `/itemsadderadditions debug`

Toggles debug mode on or off.

```text
/itemsadderadditions debug
```

Permission:

```text
itemsadderadditions.command.itemsadderadditions.debug
```
