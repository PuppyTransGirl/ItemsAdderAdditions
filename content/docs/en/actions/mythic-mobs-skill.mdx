---
title: Mythic Mobs Skill
description: Cast any MythicMobs skill at configurable power when an ItemsAdder item event fires. Integrates ItemsAdderAdditions with the MythicMobs plugin.
version: "1.0.3"
---

```yaml
mythic_mobs_skill:
  skill: "mega_attack"
  power: 0.75 # optional, default: 1.0
```

## Parameters

### `skill`

Required. The name of the MythicMobs skill to cast.

### `power`

Optional. Defaults to `1.0`. The power multiplier passed to the skill.
