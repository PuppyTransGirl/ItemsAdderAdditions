---
title: Message
description: Send a chat message to a player on any ItemsAdder event.
version: "1.0.0"
---

```yaml
message:
  text: "<rainbow>My custom message sent to %player_name%"
```

## Parameters

### `text`

Required. The message to send. Supports [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/), [PlaceholderAPI](https://modrinth.com/plugin/placeholderapi), and font images.
