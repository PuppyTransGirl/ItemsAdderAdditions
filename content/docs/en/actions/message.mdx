---
title: Message
description: Send a MiniMessage chat message from an ItemsAdder action.
version: "1.0.0"
---

```yaml
message:
  text: "<rainbow>My custom message sent to %player_name%"
```

## Parameters

### `text`

Required. The message to send. Supports [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/), [PlaceholderAPI](/en/docs/compatibility/placeholderapi), and font images.
