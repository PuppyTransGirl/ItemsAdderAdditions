---
title: Veinminer
description: Break a vein of connected same-type blocks in a single swing using an ItemsAdder veinminer action. Configurable block limit per use.
version: "1.0.0"
---

```yaml
veinminer:
  max_blocks: 16
```

## Parameters

### `max_blocks`

Required. The maximum number of blocks to break in one vein-mining action.
