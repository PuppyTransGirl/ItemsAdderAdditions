---
title: Ignite
description: Set a player or nearby entity on fire for a configurable duration using an ItemsAdder action. Duration is defined in ticks in your item YAML config.
version: "1.0.3"
---

```yaml
ignite:
  duration: 200 # ticks
```

## Parameters

### `duration`

Required. How long to set the target on fire, in ticks. 20 ticks = 1 second.
