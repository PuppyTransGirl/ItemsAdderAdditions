---
title: Actionbar
description: Send a MiniMessage action bar message from an ItemsAdder action.
version: "1.0.0"
---

```yaml
actionbar:
  text: "<red>My custom action bar sent to %player_name%"
```

## Parameters

### `text`

Required. The message to display. Supports [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/), [PlaceholderAPI](/en/docs/compatibility/placeholderapi), and font images.
