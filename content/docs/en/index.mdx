**ItemsAdderAdditions** extends [ItemsAdder](https://itemsadder.com/) with extra actions, behaviours, recipes, tags, and
features - configured directly inside your existing YML files. It resolves ItemsAdder content, MMOItems items, vanilla Minecraft items/blocks, vanilla tags, and ItemsAdderAdditions custom tags through the same compatibility layer. Most YAML-only ItemsAdderAdditions changes reload with `/iareload`; components need `/iazip`.

## Requirements

- **ItemsAdder** (latest recommended)
- **Paper** with a supported ItemsAdderAdditions NMS adapter
- **MMOItems** *(optional, only required for `mmoitems:type:id` references)*

## Compatibility

See [Compatibility and IDs](/en/docs/compatibility) for supported Minecraft versions and the accepted formats for ItemsAdder IDs, MMOItems IDs, vanilla item/block IDs, recipe IDs, vanilla tags, and custom tags.
