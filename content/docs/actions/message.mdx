---
title: Message
description: Send a chat message to a player on any ItemsAdder event. Supports PlaceholderAPI and MiniMessage formatting for colors and gradients.
---

```yaml
message:
  text: "<rainbow>My custom message sent to %player_name%"
```

Supports [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/) formatting, [PlaceholderAPI](https://modrinth.com/plugin/placeholderapi) placeholders, and [ItemsAdder](https://itemsadder.com/) font images.
