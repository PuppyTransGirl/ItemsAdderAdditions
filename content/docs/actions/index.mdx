---
title: Actions
description: Trigger effects and interactions from any ItemsAdder event
icon: Hammer
---

Actions execute when an ItemsAdder event fires - right-clicks, block breaks, entity interactions, and more.
They are configured under the item or block that triggers them.

Every action supports a shared set of [parameters](/docs/actions/parameters):
permission gating, execution delays, and player targeting.
