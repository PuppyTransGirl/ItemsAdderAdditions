---
title: Actionbar
description: Send a custom action bar message to players on any ItemsAdder event.
---

```yaml
actionbar:
  text: "<red>My custom action bar sent to %player_name%"
```

## Parameters

### `text`

Required. The message to display. Supports [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/), [PlaceholderAPI](https://modrinth.com/plugin/placeholderapi), and font images.
