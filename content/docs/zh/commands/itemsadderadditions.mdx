## `/itemsadderadditions`

ItemsAdderAdditions 的主要管理命令

## 权限

基础权限：

```text
itemsadderadditions.command.itemsadderadditions
```

## 子命令

### `/itemsadderadditions reload`

重载 ItemsAdderAdditions 的 `config.yml`，并重新应用功能、动作、行为和组件开关

```text
/itemsadderadditions reload
```

权限：

```text
itemsadderadditions.command.itemsadderadditions.reload
```

此命令不会重载 ItemsAdder 内容 YAML 文件。

修改 ItemsAdderAdditions 内容 YAML（例如动作、行为、自定义标签、配方、进度、自定义画、Valhalla 元数据或世界生成条目）后请使用 `/iareload`。

修改 `components` 节或资源包资源后请使用 `/iazip`。

### `/itemsadderadditions debug`

开启或关闭调试模式

```text
/itemsadderadditions debug
```

权限：

```text
itemsadderadditions.command.itemsadderadditions.debug
```
