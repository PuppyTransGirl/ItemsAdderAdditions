## `/itemsadderadditions`

ItemsAdderAdditions 的主管理员命令。

别名：

```text
/iaadditions
/iaa
```

## 权限

基础权限：

```text
itemsadderadditions.command.itemsadderadditions
```

## 子命令

### `/itemsadderadditions reload`

重载 ItemsAdderAdditions `config.yml`，并刷新插件级功能、动作、behaviour 和 component 开关。

```text
/itemsadderadditions reload
```

权限：

```text
itemsadderadditions.command.itemsadderadditions.reload
```

该命令**不会**重载 ItemsAdder 内容 YAML 文件。

修改 ItemsAdderAdditions 内容 YAML 节后，请使用 `/iareload`，例如动作、behaviours、自定义标签、配方、进度、自定义画、Valhalla 元数据或世界生成条目。

修改 `components` 节或资源包资源后，请使用 `/iazip`。

### `/itemsadderadditions debug`

开启或关闭 debug 模式。

```text
/itemsadderadditions debug
```

权限：

```text
itemsadderadditions.command.itemsadderadditions.debug
```
