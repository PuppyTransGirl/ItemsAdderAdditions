## `/itemsadderadditions`

ItemsAdderAdditions 的主要管理命令

## 权限

基础权限：

```text
itemsadderadditions.command.itemsadderadditions
```

## 子命令

### `/itemsadderadditions reload`

重载 ItemsAdderAdditions

```text
/itemsadderadditions reload
```

权限：

```text
itemsadderadditions.command.itemsadderadditions.reload
```

### `/itemsadderadditions debug`

开启或关闭调试模式

```text
/itemsadderadditions debug
```

权限：

```text
itemsadderadditions.command.itemsadderadditions.debug
```
