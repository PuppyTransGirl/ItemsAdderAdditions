兼容性页面说明 ItemsAdderAdditions 如何与 ItemsAdder 以及可选插件协作。当你需要确认某个集成是否必需、外部插件会怎样影响功能，或某个字段接受哪种 ID 语法时，请查看本节。

## 支持的集成

**必需：**

- [ItemsAdder](/zh/docs/compatibility/itemsadder) - 内容加载、自定义物品堆叠、自定义方块/家具、物品修改 hook 和重载事件。

**可选：**

- [MMOItems](/zh/docs/compatibility/mmoitems) - `mmoitems:type:id` 物品引用和运行时物品匹配。
- [ValhallaMMO & ValhallaTrinkets](/zh/docs/compatibility/valhalla) - 写入 ItemsAdder 物品堆叠的 Valhalla 元数据。
- [WorldGuard](/zh/docs/compatibility/worldguard) - 用于受支持 behaviours、自定义画和动作的区域 flag。
- [CoreProtect](/zh/docs/compatibility/coreprotect) - storage 打开交互日志。
- [PlaceholderAPI](/zh/docs/compatibility/placeholderapi) - 在受支持文本字段中解析 placeholder。
- [MythicMobs](/zh/docs/compatibility/mythicmobs) - `mythic_mobs_skill` 动作。

## ID 语法

物品、方块、家具、配方、标签和 namespaced ID 格式请见 [ID](/zh/docs/compatibility/ids)。
