---
title: 消息
description: 当任意 ItemsAdder 物品事件触发时，向玩家发送 MiniMessage 格式的聊天消息。支持 PlaceholderAPI 占位符
version: "1.0.0"
---

```yaml
message:
  text: "<rainbow>我的自定义消息已发送给 %player_name%"
```

## 参数

### `text`

必填。要发送的消息。支持 [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/)、[PlaceholderAPI](https://modrinth.com/plugin/placeholderapi) 和字体图片
