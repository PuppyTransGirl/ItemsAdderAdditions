---
title: 接触伤害
description: 对接触 ItemsAdder 自定义方块或家具的玩家造成伤害。可配置伤害量、冷却间隔以及中毒或凋零等药水效果
version: "1.0.0"
---

```yaml
behaviours:
  contact_damage:
    amount: 1.0
    interval: 10 # 可选，默认：20 刻
    fire_duration: 60 # 可选，默认：0 刻
    damage_when_sneaking: false # 可选，默认：true
    block_faces: # 可选
      top: true # 可选，默认：true
      north: false # 可选，默认：true
      south: false # 可选，默认：true
      west: false # 可选，默认：true
      east: false # 可选，默认：true
    potion_effect: # 可选
      type: NAUSEA
      amplifier: 0 # 可选，默认：0（0 = 等级 1）
      duration: 200 # 可选，默认：40 刻
      ambient: false # 可选，默认：false
      particles: false # 可选，默认：true
      icon: false # 可选，默认：true
    # 添加另一个效果
    potion_effect_1:
      ...
```

`type` 请使用 [PotionEffectType](https://jd.papermc.io/paper/org/bukkit/potion/PotionEffectType.html#field-summary) 中的值
