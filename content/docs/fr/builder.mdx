Choisissez une action, un comportement ou une recette, remplissez les champs, puis copiez le YAML généré directement dans votre configuration d’objet.

<Builder/>

- **Actions** : collez le résultat dans la section `events` de votre objet. Consultez la page [paramètres des actions](/fr/docs/actions/parameters) pour un exemple complet.
- **Comportements** : collez le résultat directement sous la définition de votre objet.
- **Recettes** : collez le résultat dans n’importe quel fichier YML de contenu ItemsAdder (le bloc complet `info` + `recipes` est inclus).
