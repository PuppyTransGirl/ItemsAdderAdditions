---
title: Tailleur de pierre
description: Ajouter des recettes de tailleur de pierre pour des objets ItemsAdder, MMOItems ou vanilla, avec sons et commandes optionnels à la fabrication.
version: "1.0.3"
---

Ajoutez des recettes de tailleur de pierre sans créer de fichiers de recettes supplémentaires.

## Paramètres

### `enabled`

Optionnel. Par défaut : `true`. Définissez à `false` pour ignorer l'enregistrement de la recette.

### `ingredient.item`

Obligatoire. L'objet d'entrée affiché dans le tailleur de pierre. Utilisez un matériau vanilla, un objet ItemsAdder ou un objet MMOItems.

```yaml
ingredient:
  item: STONE
```

```yaml
ingredient:
  item: my_pack:rough_marble
```

```yaml
ingredient:
  item: mmoitems:material:rough_marble
```

### `result.item`

Obligatoire. L'objet en sortie.

```yaml
result:
  item: STONE_BRICKS
```

### `result.amount`

Optionnel. Par défaut : `1`. Le nombre d’objets produits.

## Actions d'achèvement

Utilisez `on_complete` pour jouer un son et/ou exécuter des commandes lorsque la recette est fabriquée.

```yaml
on_complete:
  sound:
    name: "minecraft:entity.player.levelup"
    source: master # optionnel, par défaut : master
    volume: 1.0 # optionnel, par défaut : 1.0
    pitch: 1.0 # optionnel, par défaut : 1.0
  commands:
    reward:
      command: "give {player} gold_ingot 4"
      as_console: true # optionnel, par défaut : false
```

`{player}` est remplacé par le nom du joueur qui fabrique la recette. Les commandes sont exécutées par le joueur par défaut ; définissez `as_console: true` pour les exécuter depuis la console.

## Exemple complet

```yaml
info:
  namespace: my_pack

recipes:
  stonecutter:
    marble_tiles:
      ingredient:
        item: my_pack:rough_marble
      result:
        item: my_pack:marble_tiles
        amount: 2
      on_complete:
        sound:
          name: "minecraft:block.stonecutter.use"
        commands:
          reward:
            command: "give {player} diamond 1"
            as_console: true
```

## Notes

- Prend en charge les objets vanilla, ItemsAdder et MMOItems.
- Les tags comme `#minecraft:stone_tool_materials` ne sont pas pris en charge ici, car le tailleur de pierre nécessite une pile d’entrée concrète.
- Les ID de recette comme `marble_tiles` sont des identifiants internes.
- Le namespace provient de `info.namespace`.
