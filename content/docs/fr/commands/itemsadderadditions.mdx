## `/itemsadderadditions`

Commande administrative principale d'ItemsAdderAdditions.

## Permissions

Permission de base :

```text
itemsadderadditions.command.itemsadderadditions
```

## Sous-commandes

### `/itemsadderadditions reload`

Recharge ItemsAdderAdditions.

```text
/itemsadderadditions reload
```

Permission :

```text
itemsadderadditions.command.itemsadderadditions.reload
```

### `/itemsadderadditions debug`

Active ou désactive le mode debug.

```text
/itemsadderadditions debug
```

Permission :

```text
itemsadderadditions.command.itemsadderadditions.debug
```
