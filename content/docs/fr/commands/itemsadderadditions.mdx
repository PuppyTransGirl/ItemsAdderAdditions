## `/itemsadderadditions`

Commande d'administration principale pour ItemsAdderAdditions.

Alias :

```text
/iaadditions
/iaa
```

## Permissions

Permission de base :

```text
itemsadderadditions.command.itemsadderadditions
```

## Sous-commandes

### `/itemsadderadditions reload`

Recharge le `config.yml` d'ItemsAdderAdditions et actualise les toggles de fonctionnalités, actions, comportements et composants au niveau du plugin.

```text
/itemsadderadditions reload
```

Permission :

```text
itemsadderadditions.command.itemsadderadditions.reload
```

Cette commande ne recharge **pas** les fichiers YAML de contenu ItemsAdder.

Utilisez `/iareload` après avoir modifié des sections YAML de contenu ItemsAdderAdditions comme les actions, comportements, tags personnalisés, recettes, progrès, peintures personnalisées, métadonnées Valhalla ou entrées de génération du monde.

Utilisez `/iazip` après avoir modifié les sections `components` ou des assets du resource pack.

### `/itemsadderadditions debug`

Active ou désactive le mode debug.

```text
/itemsadderadditions debug
```

Permission :

```text
itemsadderadditions.command.itemsadderadditions.debug
```
