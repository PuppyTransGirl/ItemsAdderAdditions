## `/itemsadderadditions`

Commande administrative principale d’ItemsAdderAdditions.

## Permissions

Permission de base :

```text
itemsadderadditions.command.itemsadderadditions
```

## Sous-commandes

### `/itemsadderadditions reload`

Recharge le `config.yml` d'ItemsAdderAdditions et réapplique les toggles de fonctionnalités, actions, comportements et composants.

```text
/itemsadderadditions reload
```

Permission :

```text
itemsadderadditions.command.itemsadderadditions.reload
```

Cette commande ne recharge pas les fichiers YAML de contenu ItemsAdder.

Utilisez `/iareload` après avoir modifié les sections YAML ItemsAdderAdditions comme les actions, comportements, tags personnalisés, recettes, progrès, peintures personnalisées, métadonnées Valhalla ou entrées de génération du monde.

Utilisez `/iazip` après avoir modifié les sections `components` ou les assets du pack de ressources.

### `/itemsadderadditions debug`

Active ou désactive le mode debug.

```text
/itemsadderadditions debug
```

Permission :

```text
itemsadderadditions.command.itemsadderadditions.debug
```
