**ItemsAdderAdditions** étend [ItemsAdder](https://itemsadder.com/) avec des actions, comportements et
fonctionnalités supplémentaires, configurées directement dans vos fichiers YML existants. Il résout maintenant le contenu ItemsAdder, les objets MMOItems, les objets/blocs vanilla et les tags vanilla avec la même couche de compatibilité. Pas de redémarrage, juste `/iareload`.

## Prérequis

- **ItemsAdder** (dernière version recommandée)
- **Paper** 1.20.6+
- **MMOItems** *(optionnel, requis uniquement pour les références `mmoitems:type:id`)*

## Compatibilité

Consultez [Compatibilité et IDs](/docs/compatibility) pour les formats acceptés des IDs ItemsAdder, IDs MMOItems, IDs d’objets/blocs vanilla et tags vanilla.
