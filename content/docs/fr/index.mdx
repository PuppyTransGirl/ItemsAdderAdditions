**ItemsAdderAdditions** étend [ItemsAdder](https://itemsadder.com/) avec des actions, comportements et
fonctionnalités supplémentaires, configurées directement dans vos fichiers YML existants. Il résout maintenant le contenu ItemsAdder, les objets MMOItems, les objets/blocs vanilla, les tags vanilla et les tags personnalisés ItemsAdderAdditions avec la même couche de compatibilité. La plupart des changements YAML ItemsAdderAdditions se rechargent avec `/iareload` ; les composants nécessitent `/iazip`.

## Prérequis

- **ItemsAdder** (dernière version recommandée)
- **Paper** avec un adaptateur NMS ItemsAdderAdditions pris en charge
- **MMOItems** *(optionnel, requis uniquement pour les références `mmoitems:type:id`)*

## Compatibilité

Consultez [Compatibilité et IDs](/fr/docs/compatibility) pour les versions Minecraft prises en charge et les formats acceptés des IDs ItemsAdder, IDs MMOItems, IDs d’objets/blocs vanilla, tags vanilla et tags personnalisés.
