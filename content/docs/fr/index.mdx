**ItemsAdderAdditions** étend [ItemsAdder](https://itemsadder.com/) avec des actions, comportements, recettes, tags et
fonctionnalités supplémentaires, configurés directement dans vos fichiers YML existants. Il résout le contenu ItemsAdder, les objets MMOItems, les objets/blocs Minecraft vanilla, les tags vanilla et les tags personnalisés ItemsAdderAdditions avec la même couche de compatibilité. La plupart des changements ItemsAdderAdditions uniquement en YAML se rechargent avec `/iareload` ; les composants nécessitent `/iazip`.

## Prérequis

- **ItemsAdder** (dernière version recommandée)
- **Paper** avec un adaptateur NMS ItemsAdderAdditions pris en charge
- **MMOItems** *(optionnel, requis uniquement pour les références `mmoitems:type:id`)*

## Compatibilité

Consultez [Compatibilité](/fr/docs/compatibility) pour les notes d'intégration et [IDs](/fr/docs/compatibility/ids) pour les IDs ItemsAdder, IDs MMOItems, IDs d'objets/blocs vanilla, IDs de recettes, tags vanilla et tags personnalisés.
