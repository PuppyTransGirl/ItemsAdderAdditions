---
title: Dégâts au contact
description: Infliger des dégâts aux joueurs qui touchent un bloc ou meuble personnalisé ItemsAdder. Montant des dégâts, délai de recharge et type de protection configurables.
version: "1.0.0"
---

```yaml
behaviours:
  contact_damage:
    amount: 1.0
    interval: 10 # optionnel, défaut : 20 ticks
    fire_duration: 60 # optionnel, défaut : 0 ticks
    damage_when_sneaking: false # optionnel, défaut : true
    block_faces: # optionnel
      top: true # optionnel, défaut : true
      north: false # optionnel, défaut : true
      south: false # optionnel, défaut : true
      west: false # optionnel, défaut : true
      east: false # optionnel, défaut : true
    potion_effect: # optionnel
      type: NAUSEA
      amplifier: 0 # optionnel, défaut : 0 (0 = niveau 1)
      duration: 200 # optionnel, défaut : 40 ticks
      ambient: false # optionnel, défaut : false
      particles: false # optionnel, défaut : true
      icon: false # optionnel, défaut : true
    # Ajouter un autre effet
    potion_effect_1:
      ...
```

Pour `type`, utilisez une valeur depuis [PotionEffectType](https://jd.papermc.io/paper/org/bukkit/potion/PotionEffectType.html#field-summary).
