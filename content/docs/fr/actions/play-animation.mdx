---
title: Jouer une animation
description: Déclencher une animation de meuble depuis n'importe quel événement ItemsAdder.
version: "1.0.0"
---

```yaml
play_animation:
  name: "spin"
```

## Paramètres

### `name`

Obligatoire. Le nom de l'animation à jouer, tel que défini dans la configuration du meuble.
