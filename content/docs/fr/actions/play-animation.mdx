---
title: Jouer une animation
description: Déclencher une animation de meuble ItemsAdder nommée lorsqu’un événement d’objet se déclenche. Indiquez le nom de l’animation directement dans votre YAML d’action.
version: "1.0.0"
---

```yaml
play_animation:
  name: "spin"
```

## Paramètres

### `name`

Obligatoire. Le nom de l'animation à jouer, tel que défini dans la configuration du meuble.
