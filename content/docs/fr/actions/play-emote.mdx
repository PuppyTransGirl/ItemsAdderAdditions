---
title: Jouer une emote
description: Faire exécuter une emote ItemsAdder nommée à un joueur lorsqu’un événement d’objet se déclenche. Indiquez le nom de l’emote directement dans votre configuration d’action.
version: "1.0.0"
---

```yaml
play_emote:
  name: "wave"
```

## Paramètres

### `name`

Obligatoire. Le nom de l'emote à jouer.
