---
title: Message
description: Envoyer un message de chat formaté avec MiniMessage à un joueur lorsqu’un événement d’objet ItemsAdder se déclenche. Prend en charge les placeholders PlaceholderAPI.
version: "1.0.0"
---

```yaml
message:
  text: "<rainbow>Mon message personnalisé envoyé à %player_name%"
```

## Paramètres

### `text`

Obligatoire. Le message à envoyer. Prend en charge [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/), [PlaceholderAPI](https://modrinth.com/plugin/placeholderapi) et les images de police.
