---
title: Message
description: Envoyer un message dans le chat lors de n'importe quel événement ItemsAdder.
version: "1.0.0"
---

```yaml
message:
  text: "<rainbow>Mon message personnalisé envoyé à %player_name%"
```

## Paramètres

### `text`

Obligatoire. Le message à envoyer. Supporte [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/), [PlaceholderAPI](https://modrinth.com/plugin/placeholderapi) et les images de police.
