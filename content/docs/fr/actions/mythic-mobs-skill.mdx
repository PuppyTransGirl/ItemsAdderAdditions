---
title: Compétence MythicMobs
description: Lancer n’importe quelle compétence MythicMobs avec une puissance configurable lorsqu’un événement d’objet ItemsAdder se déclenche. Intègre ItemsAdderAdditions à MythicMobs.
version: "1.0.3"
---

```yaml
mythic_mobs_skill:
  skill: "mega_attack"
  power: 0.75 # optionnel, défaut : 1.0
```

## Paramètres

### `skill`

Obligatoire. Le nom de la compétence MythicMobs à lancer.

### `power`

Optionnel. Par défaut : `1.0`. Le multiplicateur de puissance transmis à la compétence.
