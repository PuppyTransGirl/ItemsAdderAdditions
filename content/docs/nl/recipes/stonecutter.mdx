Voeg steensnijderrecepten toe zonder extra receptbestanden aan te maken.

## Parameters

### `enabled`

Optioneel. Standaard: `true`. Stel dit in op `false` om het registreren van het recept over te slaan.

### `ingredient.item`

Verplicht. Het invoeritem dat in de steensnijder wordt getoond. Gebruik een vanilla materiaal, een ItemsAdder-item of een MMOItems-item.

```yaml
ingredient:
  item: STONE
```

```yaml
ingredient:
  item: my_pack:rough_marble
```

```yaml
ingredient:
  item: mmoitems:material:rough_marble
```

### `result.item`

Verplicht. Het uitvoeritem.

```yaml
result:
  item: STONE_BRICKS
```

### `result.amount`

Optioneel. Standaard: `1`. Het aantal geproduceerde items.

## Voltooiingsacties

Gebruik `on_complete` om een geluid af te spelen en/of commando's uit te voeren wanneer het recept wordt gemaakt.

```yaml
on_complete:
  sound:
    name: "minecraft:entity.player.levelup"
    source: master # optioneel, standaard: master
    volume: 1.0 # optioneel, standaard: 1.0
    pitch: 1.0 # optioneel, standaard: 1.0
  commands:
    reward:
      command: "give {player} gold_ingot 4"
      as_console: true # optioneel, standaard: false
```

`{player}` wordt vervangen door de naam van de speler die het recept maakt. Commando's worden standaard als speler uitgevoerd; stel `as_console: true` in om ze vanuit de console uit te voeren.

## Volledig voorbeeld

```yaml
info:
  namespace: my_pack

recipes:
  stonecutter:
    marble_tiles:
      ingredient:
        item: my_pack:rough_marble
      result:
        item: my_pack:marble_tiles
        amount: 2
      on_complete:
        sound:
          name: "minecraft:block.stonecutter.use"
        commands:
          reward:
            command: "give {player} diamond 1"
            as_console: true
```

## Opmerkingen

- Ondersteunt vanilla, ItemsAdder- en MMOItems-items.
- Tags zoals `#minecraft:stone_tool_materials` worden hier niet ondersteund, omdat de steensnijder één concrete invoerstack nodig heeft.
- Recept-ID's zoals `marble_tiles` zijn interne identificatoren.
- De namespace komt van `info.namespace`.
