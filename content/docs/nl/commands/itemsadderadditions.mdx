## `/itemsadderadditions`

Hoofd-beheerscommando voor ItemsAdderAdditions.

## Toestemmingen

Basistoestemming:

```text
itemsadderadditions.command.itemsadderadditions
```

## Subcommando's

### `/itemsadderadditions reload`

Herlaadt ItemsAdderAdditions.

```text
/itemsadderadditions reload
```

Toestemming:

```text
itemsadderadditions.command.itemsadderadditions.reload
```

### `/itemsadderadditions debug`

Schakelt debugmodus in of uit.

```text
/itemsadderadditions debug
```

Toestemming:

```text
itemsadderadditions.command.itemsadderadditions.debug
```
